# Path: Investor Research Agents

Goal: build agent workflows for evidence collection, thesis checks, diligence, memos, and risk registers.

## Primary Seeds

- [TradingAgents](https://github.com/TauricResearch/TradingAgents)
- [ai-hedge-fund](https://github.com/virattt/ai-hedge-fund)
- [FinRobot](https://github.com/AI4Finance-Foundation/FinRobot)
- [FinGPT](https://github.com/AI4Finance-Foundation/FinGPT)

## Workflow Shape

```text
source question -> evidence collection -> thesis memo -> risk register -> human gate
```

## Required Gates

- Evidence date.
- Source URL.
- Confidence level.
- Risk section.
- Human decision line.
- Non-advisory boundary.

## Not In Scope

- Automated investments.
- Return promises.
- Legal, tax, or securities conclusions.
